# Техническое задание

## Восстановление зависимости вклада двутавра в несущую способность балки методами машинного обучения и биоинспирированной оптимизации

## 1. Цель работы

Построить и сравнить набор современных методов машинного обучения, биоинспирированной оптимизации и вывода символьных формул для задачи предсказания вклада двутавра $Q_\text{дв}$ в несущую способность композитной/стальной балки по её характеристикам.

Работа носит исследовательский характер. Целью является не достижение максимальной точности одним методом, а системный охват веера подходов и обоснованное сравнение их применимости. Отрицательный результат отдельного метода считается полноценным результатом при условии корректного анализа причин.

Работа задумана как современное развитие методики нейросетевого моделирования с выводом инженерной формулы регрессии, с заменой ручного подбора на автоматизированные и более современные подходы.

## 2. Исходные данные

Таблица образцов двутавровых балок (файл `Таблица_с_балками_КубГУ.xlsx`).

**Входные характеристики:**

- относительный пролёт среза $a/h_0$;
- тип материала двутавра (композитный / стальной);
- высота двутавра $H$, мм;
- толщина стенки $s$, мм;
- предел прочности $R$, МПа;
- модуль упругости $E$, МПа.

**Целевая величина:**

- $Q_\text{дв}$ — вклад двутавра в поперечную силу, кН (в двух вариантах: по СП 63.13330 и по руководству 1978 г.).

## 3. Стратегия работы с данными

### 3.1. Малая выборка

Все модели проверяются по схеме Leave-One-Group-Out по уникальным профилям: обучение на 5 профилях, предсказание отложенного 6-го. Обычное train/test-разбиение неприменимо: дало бы завышенную точность из-за дублей.

### 3.2. Синтез дополнительных данных

Целевая величина $Q_\text{дв}$ при синтезе не пересчитывается. Обоснование: зависимость гладкая, поэтому при малом изменении входных параметров $Q_\text{дв}$ меняется незначительно и принимается прежним. В каждую группу $a/h_0$ добавляется необходимое число синтезированных образцов.

**Диапазоны возмущения входных параметров:**

- высота $H$: ± несколько мм;
- толщина $s$: ± десятые доли мм;
- прочность $R$: ± несколько МПа;
- модуль упругости $E$: ± 50 МПа.

**Условия корректности:**

Разброс входов удерживается малым (в указанных пределах). Приближение справедливо только для малых возмущений; при больших сдвигах входов сохранение прежнего $Q_\text{дв}$ дало бы физически недостоверные данные.

Назначение синтеза — повышение устойчивости моделей к шуму и обеспечение достаточного объёма для методов, чувствительных к размеру выборки (нейросети). Синтез не заполняет пробелы между исходными образцами: обобщающая способность в промежутках определяется исходными 6 профилями.

## 4. Методы

Методы сгруппированы по назначению.

### 4.1. Базовая линия

- **Линейная регрессия** — планка снизу; сложные методы, не превосходящие её, признаются неоправданными.
- **Регуляризованная линейная** (Ridge, Lasso, ElasticNet) — устойчивость на малой выборке; Lasso дополнительно выполняет отбор признаков.

### 4.2. Предсказательные модели («чёрный ящик»)

- **Градиентный бустинг** — ансамбль деревьев (ожидаемо слаб на малой выборке; результат информативен).
- **Метод ближайших соседей** (KNN).
- **Метод опорных векторов** (SVR).
- **Gaussian Process Regression** (GPR) — предсказание с оценкой неопределённости (доверительный коридор), современный и подходящий для малых данных подход.

### 4.3. Вывод явной формулы

Прямая современная замена ручного подбора степенной формулы алгоритмом Левенберга–Марквардта:

- **Symbolic Regression** (генетическое программирование; gplearn / PySR) — автоматический поиск структуры и коэффициентов формулы.
- **Биоинспирированный подбор коэффициентов** заданной (например степенной) формы — генетический алгоритм, PSO, дифференциальная эволюция, CMA-ES. Сравнение оптимизаторов между собой.

### 4.4. Физически-информированные и современные нейросети

- **Физически-информированные нейронные сети** (PINN) — встраивание физических ограничений в обучение.
- **Многослойный перцептрон** с регуляризацией/dropout — сравнение с нейросетью предыдущего поколения.

### 4.5. Экспериментальный блок

Раздел для новейших подходов; ожидаемо высокий риск неудачи, что допустимо и ценно:

- **Reinforcement Learning** (actor-critic) — исследование применимости к задаче регрессии; вероятная демонстрация избыточности метода с обоснованием границ.
- **Трансформерная / байесовская символьная регрессия.**
- **TabPFN** — модель под малые табличные выборки.

## 5. Метрики и сравнение

Для сопоставимости с методикой диссертации:

- отношение $Q_\text{эксп} / Q_\text{пред}$ (в идеале ~ 1) и коэффициент вариации;
- дополнительно: MAE, RMSE, $R^2$ — для полноты картины.

**Результат работы:** сводная таблица «метод × метрики» + отдельно выведенные символьные формулы + анализ применимости каждого метода (включая неуспешные).

## 6. Этапы и порядок

1. Подготовка данных: парсинг, чистка, анализ признаков.
2. Реализация синтеза данных (малые возмущения входов, раздел 3.2).
3. Базовая линия + классический ML (скелет пайплайна, первые метрики).
4. Ядро новизны: вывод формулы (symbolic regression + биоинспирированные).
5. Современные предсказатели (GPR, PINN, нейросети).
6. Экспериментальный блок (по остаточному приоритету).
7. Сведение результатов, оформление разделов под каждый метод.

Порядок построен так, чтобы на каждом шаге существовал показуемый промежуточный результат.

**Формат отчётности:** регулярные короткие отчёты о ходе работы; согласование в общей беседе (Telegram) с руководителем.
